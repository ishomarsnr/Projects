Decision Tree Application for Phishing Detection

Introduction

Phishing is a significant cybersecurity threat that involves tricking individuals into revealing sensitive information by pretending to be a trustworthy entity. In this project, I built a decision tree model to detect phishing attempts based on various features extracted from URLs.

Data Generation

I generated a simulated dataset to include features commonly associated with phishing attempts, such as URL length, use of IP address, presence of HTTPS token, age of domain, and having a subdomain. 


import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import accuracy_score
from sklearn.tree import plot_tree
import matplotlib.pyplot as plt

# Generate a simulated dataset for phishing detection
np.random.seed(42)
data_size = 1000
url_length = np.random.choice([-1, 0, 1], data_size)
using_ip_address = np.random.choice([-1, 1], data_size)
https_token = np.random.choice([-1, 1], data_size)
age_of_domain = np.random.choice([-1, 1], data_size)
having_sub_domain = np.random.choice([-1, 0, 1], data_size)
phishing_attempt = np.random.choice([-1, 1], data_size)

df = pd.DataFrame({
    'URL Length': url_length,
    'Using IP Address': using_ip_address,
    'HTTPS Token': https_token,
    'Age of Domain': age_of_domain,
    'Having Sub Domain': having_sub_domain,
    'Phishing Attempt': phishing_attempt
})

df.head()

Model Training and Evaluation

The dataset was split into training and testing sets. I trained a Decision Tree classifier on the training set and evaluated its performance on the test set. 


# Split the dataset into training and testing sets
X = df.drop('Phishing Attempt', axis=1)
y = df['Phishing Attempt']
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Train a Decision Tree classifier
clf_entropy = DecisionTreeClassifier(criterion='entropy', random_state=42)
clf_entropy.fit(X_train, y_train)
predictions = clf_entropy.predict(X_test)
accuracy = accuracy_score(y_test, predictions)
print(f"Accuracy: {accuracy * 100:.2f}%")

Model Visualization

The trained Decision Tree model is visualized to show how decisions are made based on the features. 


# Visualize the trained Decision Tree
clf = DecisionTreeClassifier(criterion='entropy', max_depth=3, random_state=42)
clf.fit(X_train, y_train)

plt.figure(figsize=(20,10))
plot_tree(clf, filled=True, feature_names=X.columns.tolist(), class_names=['Legitimate', 'Phishing'], rounded=True, fontsize=14)
plt.savefig('assets/img/decision_tree_phishing.png')
plt.show()

Conclusion

The Decision Tree visualization shows how decisions are made based on the features. Each node represents a decision point where the dataset is split based on a feature's value, and the branches represent the outcome of that decision.

  • The condition (e.g., URL Length <= 0.5) used to split the data.
  • The Gini impurity, a measure of the node's purity (0 indicates all samples in the node belong to a single class).
  • The number of samples that fall into each category at that point.
  • The class distribution (value), showing how many samples from each class reach that node.
  • The color of the nodes may indicate the predominant class, with different shades representing the proportion of each class.
